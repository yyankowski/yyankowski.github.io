Useful commands and cheatsheets for everyday life of a fullstack engineer.

Git

Cleanup branches with deleted remotes: git fetch --all --prune && git branch --v | grep "\[gone\]" | awk '{print $1}' | xargs git branch -D

Reset master to match remote: git fetch --all && git reset --hard origin/master

Pull only with fast forward: git config --global pull.ff only

Remove unstaged files:

Display files to be removed: git clean -n

Do the actual deletion: git clean -f

Trunk-based Development:

Pulling remote: git pull --rebase

Pushing to remote: git pull --rebase && git push
